quantité : 3 exemplaires, année d'achat 2019

remarque : batterie un peu volumineuse mais permet de réaliser plusieurs séries de match sans changer.

Attention ces batteries sont fragiles. Il ne surtout pas les décharger en desous d'un seuil (22.5v) sinon
elles seront abimées de façon irreversible. Bien regarder la tension entre chaque match ou acheter un voltmère
embarqué.

Lorsque vous n'utilisez pas les batteries (vacances, etc) il faut mettre la batterie en mode stockage soit 3.8v par élement.

Les charger avec un chargeur adapté sinon Boummm.
